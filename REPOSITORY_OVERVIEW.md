# map-tool リポジトリ確認メモ

## 構成
- `index.html`:
  - メイン画面（住所入力、半径ボタン、出力欄、2画面マップ）を定義。
- `script.js`:
  - APIキー管理、Google Maps 初期化、住所ジオコーディング、ピン配置、半径円描画、定型文生成、コピー処理を実装。
  - 設定（自動半径計算・配置切替）の永続化ロジックを実装。
- `help.html`:
  - APIキー取得手順を案内するヘルプ画面。

## 現在の業務仕様との整合
- 出力文は `{緯度, 経度}で◯◯mピン（消防出動情報向けのメッセージです）` 形式で生成。
- ピン半径の選択肢は `50m / 100m / 300m / 500m / 1km / 不可` の固定仕様。
- 自動半径計算でも上記プリセット内で丸める実装で、1kmを超える場合は 1km 扱い。

## 補足
- APIキーは `localStorage` に保存され、初回のみ入力ダイアログを表示。
- 設定メニューの「自動半径計算」「配置を切り替える」は `localStorage` で保持され、再読込後も維持。
- `index.html`: メイン画面。住所入力、半径ボタン（50/100/300/500/1km/不可）、出力欄、作業用地図と参照用地図（埋め込み）を表示。
- `script.js`: APIキー管理、Google Maps 初期化、住所ジオコーディング、ピン配置、半径円描画、定型文生成、コピー処理などのロジック。
- `help.html`: APIキー取得手順のヘルプ画面。

## 現在の業務仕様との整合
- 出力文は `{"緯度, 経度"}` 形式を含む消防出動情報向けメッセージを生成。
- ピン半径の選択肢は `50m / 100m / 300m / 500m / 1km / 不可`。
- 自動半径計算でも上記プリセット内で丸める実装になっており、1kmを超える場合は1kmに設定される。

## 補足
- APIキーは `localStorage` に保存され、初回のみ入力ダイアログを表示。
- 参照地図は Google Maps Embed API を利用。
- 利用回数の目安表示（本日/今月残り）を `localStorage` で管理。
