住所抽出ツール

CSV住所データベースを貼り付けると、スプレッドシートにそのまま貼り付けられる3列形式で住所を抽出します。
← 座標取得ツールへ戻る
入力CSVはヘッダー行付きでも利用できます。抽出ルールは以下のとおりです。
追加済みの境界データは、下のプルダウンから 広島市 / 倉敷市 / 京都市 / 埼玉西部 のように選んで、CSVなしで住所データを作成できます。
1列目: pref + city + ward(存在時) + oaza_cho + machiaza_typeが2なら chome_number + 「丁目」
2列目: pref + city + ward(存在時)
市町村名を指定すると、その市町村に一致する行だけ抽出します。複数指定する場合は カンマ / 読点 / セミコロン 区切りで入力できます。政令指定都市は 京都市 のように市名だけを指定しても、その市に属するすべての区をまとめて抽出できます。 3列目: 住所に○丁目が含まれていない場合のみ「丁目」
差分比較では、貼り付けたCSVから抽出した対象市区町村の住所一覧を、既存の全境界データ（広島市 / 倉敷市 / 京都市 / 埼玉西部） と照合します。
例: 広島市、倉敷市、京都市、埼玉西部
CSVファイルをドラッグ＆ドロップすると内容を自動で読み込みます。
