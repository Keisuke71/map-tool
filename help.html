📍 座標取得ツール ヘルプ

このツールを利用するために必要な「Google Maps APIキー」の取得方法と設定手順を解説します。

手順1：APIキーを取得する

Googleの地図サービスを利用するための「鍵」を取得します。以下の手順で行ってください。

  1. Google Cloud Console にアクセスします。
  2. 画面上部のプロジェクト選択部分から「新しいプロジェクト」を作成します（名前は「Map-Tool」など自由です）。
  3. メニューの「APIとサービス」＞「ライブラリ」を開き、以下の3つを検索して【有効】にしてください。
    • Maps JavaScript API
    • Geocoding API
    • Maps Embed API
  4. 「APIとサービス」＞「認証情報」を開き、「＋ 認証情報を作成」＞「APIキー」をクリックします。
  5. 表示された長い文字列（APIキー）をコピーしてください。
💡 重要：課金を防ぐ設定（推奨）
「APIとサービス」＞「割り当て」から、Geocoding API等の「1日あたりのリクエスト数」を 1000 に設定することをお勧めします。これで無料枠を超える心配がなくなります。

手順2：ツールに設定する

取得したキーをツールに入力します。

  1. ツールの初期設定画面に戻ります。
  2. 入力欄に、手順1でコピーしたAPIキーを貼り付けます。
  3. 「利用開始」ボタンを押してください。

※ 入力したキーはブラウザに保存されるため、次回からは入力不要です。

困ったときは

Q. 「エラーが発生しました」と出る

APIキーが間違っているか、Google Cloud側で「請求先アカウント」の設定が完了していない可能性があります。Google Cloud Consoleを確認してください。

Q. キーを間違えて登録してしまった

ツールの画面右上にある歯車アイコン（⚙️）をクリックし、「🔑 APIキーを削除する」を選択すると、再入力できるようになります。
