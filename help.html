📍 座標取得ツール ヘルプ

このツールを利用するために必要な「Google Maps APIキー」の取得方法を解説します。

⚠️ APIキー作成前にお読みください

GoogleのAPIキーを取得する際、会社のアカウントか個人用アカウントか迷う場合があります。以下の点にご注意ください。

項目 個人用アカウント (@gmail.com) 会社アカウント (@会社名.net)
支払い登録 ご自身のカード登録が必要 会社の管理設定に依存します
利用制限 制限なく作成可能 IT部門により禁止されている場合があります

【推奨】
2〜3名での小規模な利用で、会社側でGoogle Cloudの設定が済んでいない場合は、個人用のGoogleアカウントでの作成が最もスムーズです。月間200ドル（約3万円）分の無料枠があるため、通常の業務利用であれば料金が発生することはありませんが、無料枠内であってもクレジットカード等の登録は必須となります。

手順1：APIキーを取得する

Googleの地図サービスを利用するための「鍵」を取得します。

  1. Google Cloud Console にアクセスします。
  2. 画面上部のプロジェクト選択部分から「新しいプロジェクト」を作成します。
  3. メニューの「APIとサービス」＞「ライブラリ」を開き、以下の3つを検索して【有効】にしてください。
    • Maps JavaScript API
    • Geocoding API
    • Maps Embed API
  4. 「APIとサービス」＞「認証情報」を開き、「＋ 認証情報を作成」＞「APIキー」をクリックします。
  5. 表示されたAPIキーをコピーしてください。
💡 安全のための設定
「APIとサービス」＞「割り当て」から、各APIの「1日あたりのリクエスト数」を 1000 程度に制限しておくと、万が一の際も課金が発生せず安心です。

手順2：ツールに設定する

取得したキーをツールに入力します。

  1. ツールの初期設定画面に戻ります。
  2. 入力欄に、コピーしたAPIキーを貼り付けます。
  3. 「利用開始」ボタンを押してください。

※ キーはブラウザに保存されるため、次回から入力は不要です。
