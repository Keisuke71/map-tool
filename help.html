📍 座標取得ツール ヘルプ

このツールを利用するために必要な「Google Maps APIキー」の取得方法を解説します。

⚠️ APIキー作成前にお読みください

Google APIキーは無料枠内でもクレジットカード等の支払い情報の登録が必須です。会社用アカウントで行うと、支払い情報の紐付けや権限エラー等でトラブルになる可能性があります。

【推奨】
2〜3名での小規模利用の場合は、ご自身の「個人用Googleアカウント (@gmail.com)」での作成が最もスムーズです。

💡 Chromeをご利用の方へ
普段「会社のメールアドレス」でChromeにログインしている場合は、以下の手順で個人のプロファイルに切り替えてから作業を開始してください。
  1. Chrome右上のプロフィールアイコンをクリック
  2. 「他のプロファイル」から個人用のアカウントを選択するか、「ゲスト」または「追加」を選んで新しいウィンドウを開く
  3. その新しいウィンドウで Google Cloud Console にアクセスする
※これにより、誤って会社のビジネスアカウントに個人のカード情報を登録してしまう事故を防げます。

手順1：APIキーを取得する

Googleの地図サービスを利用するための「鍵」を取得します。

  1. Google Cloud Console にアクセスします。
  2. 画面上部のプロジェクト選択部分から「新しいプロジェクト」を作成します。
  3. メニューの「APIとサービス」＞「ライブラリ」を開き、以下の3つを検索して【有効】にしてください。
    • Maps JavaScript API
    • Geocoding API
    • Maps Embed API
  4. 「APIとサービス」＞「認証情報」を開き、「＋ 認証情報を作成」＞「APIキー」をクリックします。
💡 安全のための設定（重要）
「APIとサービス」＞「割り当て」から、各APIの「1日あたりのリクエスト数」を 1000 程度に制限しておくと、万が一の際も課金が発生せず安心です。

手順2：ツールに設定する

取得したキーをツールに入力します。

  1. ツールの初期設定画面に戻ります。
  2. 入力欄に、コピーしたAPIキーを貼り付けます。
  3. 「利用開始」ボタンを押してください。
